This Keyword

This is a keyword in js and has different value at different places.

  • 'this' in global scope points to the 'Global Object' i.e in Chrome it points to 'window'
  • 'this' in the function points to UNDEFINED in strict mode.
  • 'this' in function points to 'Gloab Object' in NON-STRICT MODE because of 'this subsitution' (if this is pointing to undefined or null it will automatically points this to global object)
  • 'this' in method points to the object it is called with.
  • 'this' in arrow functions points to its parents lexical environment.
  • 'this' in the DOM points to the element it is used with.

Call Apply and Bind

They are all used to change the reference of 'this' keyword.
